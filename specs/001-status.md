# Spec 001  /status (Hero)

## Purpose
First impression. Communicate who Mukthanand is, current availability, and current "build version" in 5 seconds. This holds the signature animated moment.

## Content
- Name + one bold positioning line (e.g. "Backend & Full-Stack Engineer").
- Status indicator: "Available  open to roles" with a green live dot (semantic green).
- Build version: `v2.0.0  Graduating July 2025` (mono, violet version tag, sparingly).
- Short human-readable subline so non-technical readers instantly get it.
- Clear nav to other sections.

## Visual / motion
- Oversized Clash Display title that settles in with a spring (shared easing).
- One cyan accent element reacting subtly to the cursor (magnetic or glow).
- Calm, huge whitespace.

## Acceptance
- Understandable to a non-technical recruiter in 5s.
- One memorable motion moment, not many.
